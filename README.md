# 🖐️ Gesture-Based Interactive Math Solver

An AI-powered educational application that enables users to solve mathematical expressions using hand gestures. The system captures gestures through a webcam, recognizes them using computer vision, converts them into mathematical expressions, and provides real-time solutions.

## 🚀 Features
- Real-time hand gesture recognition
- AI-based mathematical expression detection
- Instant calculation and result display
- Interactive and user-friendly interface
- Enhances digital learning experience

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- Machine Learning
- Computer Vision
- NumPy
- SymPy
